# Wolf3D
Recreation of Wolfenstein3D

## How to install

[Clone](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) this repository, then type the following instructions in a shell being in the clone directory:
```
make && ./wolf3d
```

## Controls

```
ZQSD Arrows - Move
AE - Rotate
F - Toggle flashlight
LeftShift - Run
Esc - Pause
```
